# Dotfiles

This repository contains my personal dotfiles for configuring various applications and tools on my system. These
configurations help streamline my workflow and maintain consistency across different environments.

# Usage
To use these dotfiles, you can clone the repository to your local machine and create symbolic links to the appropriate
locations. This can be done automatically with [stow](https://www.gnu.org/software/stow/manual/stow.html) program or manually by creating symlinks.
